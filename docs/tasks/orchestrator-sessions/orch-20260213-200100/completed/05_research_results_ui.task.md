## 📚 Context

### Data Shape

**Academic Paper:**
```typescript
{
  sourceType: 'ACADEMIC';
  title: string; authors: string; year: string;
  citationCount: number; abstractText: string;
  openAccessUrl: string | null; fileUrl: string;
  venue: string;
}
```

**Web Source:**
```typescript
{
  sourceType: 'WEB';
  title: string; snippet: string; fileUrl: string;
}
```

### Design Requirements

**Academic Paper Card:**
- Clickable title → `fileUrl` in new tab
- Authors (truncated if >3), Year + Venue
- Citation badge: "🔗 42 citations"
- Collapsible abstract (first 2 lines visible)
- "📥 Download PDF" if `openAccessUrl` exists, else "🔒 Paywalled"

**Web Source Card:**
- Clickable title → `fileUrl` in new tab
- AI snippet (1-2 sentences)
- "Open Source →" button

**Container:**
- Tabs: "📄 Papers (X)" and "🌐 Web Sources (X)"
- Sort papers by citation count descending
- Empty states, loading skeletons
