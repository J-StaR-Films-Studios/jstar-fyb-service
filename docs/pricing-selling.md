Use these scripts when potential clients ask for price ranges via WhatsApp, DM, or email.

```text
🥉 **Code & Go (₦120,000)**
Great if you just need the technical heavy lifting done.
• Complete Source Code (Bug-free)
• Setup Guide (How to run it)
• Basic Documentation
✅ Best for students who just need the working app and can handle the writing.

🥈 **Defense Ready (₦200,000) - 🔥 MOST POPULAR**
The "Peace of Mind" package. We don't just build it; we prep you to defend it.
• Everything in Code & Go
• Full Chapter 1-5 Write-up
• Defense Slide Deck (PowerPoint)
• Mock Defense Session (We drill you on likely questions)
✅ Best for students who want to guarantee they pass their defense.

🥇 **The Soft Life (₦320,000)**
Zero stress. You literally just show up.
• Everything in Defense Ready
• Priority Handling (Skip the queue)
• Unlimited Revisions
• Full Speech Script Written for you
• 1-on-1 Code Walkthrough (We teach you every line)
✅ Best for students who want a Distinction with zero headache.

Which level of support are you looking for right now?
```
